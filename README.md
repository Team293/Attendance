# Simple attendance bot

### How to run:

1) Download the latest jar from releases
2) Create a .env with the following variables:

```
SLACK_APP_TOKEN=
SLACK_BOT_TOKEN=
DB_URL=jdbc:postgresql://
DB_USER=
DB_PASSWORD=
```

and fill them in

3) Add the commands and workflows to the Slack admin dashboard
